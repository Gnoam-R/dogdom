## 🛠️ 개발 도구 및 활용 기술

- 개발 언어 : Kotlin
- 개발 환경 : Android Studio
- 디자인 툴 : Figma
- 협업 도구 : Github, Notion
- 사용 라이브러리 : Room, Hilt, ViewModel, databinding, Navigation, etc

<br/>

## 👨‍💻 프로젝트 소개
이 플랫폼은 우리의 삶에 있어 소중한 동반자, 가족인 애완동물의 소중한 순간들을 기록하고, 다른 반려견 보호자들과 소통할 수 있는 공간을 제공 합니다.
애완동물과 생활함에 있어서 다양한 정보와 팁들을 공유할 수 있습니다.
